# Meu Carro

No diretório raiz do projeto utilize esses comandos na seguinte ordem:

#### Compilação Windows
```bash
javac -d bin src\main\java\car\*.java src\main\java\*.java
```

#### Compilação Linux
```bash
javac -d bin src/main/java/car/*.java src/main/java/*.java
```

#### Execução
```bash
java -cp bin Main
```
